## Phase 1.5 — Beginner to Intermediate

> **Version:** 1.0.0

> **Target:** Both Quranic Arabic learners and conversational Arabic learners
> **Languages:** English + Urdu (user selects at onboarding)
> **Total:** 8 Units → 40 Chapters → ~180 Lessons

---

## Curriculum Philosophy

### Two Tracks, One Path
Every lesson serves two audiences simultaneously:
- **Quranic track:** Examples pulled from the Quran, Salah phrases, and Duas
- **Conversational track:** Same grammar/vocabulary applied to daily life situations

The learner does not choose a track — both are woven into every lesson. A lesson on "Ism (Noun)" teaches the word كِتَاب (book) both in a Quranic context and in a sentence like "The book is on the table."

### Progression Logic
```
Unit 1: Script & Sound         → Can read Arabic letters
Unit 2: First Words            → Knows 100 core words
Unit 3: Sentence Foundations   → Understands basic sentence structure
Unit 4: The Noun System        → Understands Ism, gender, number, case
Unit 5: The Verb System        → Understands Fi'l, tenses, conjugation
Unit 6: Particles & Connectors → Can connect sentences
Unit 7: Quranic Deep Dive      → Applies grammar to Quran directly
Unit 8: Conversation Builder   → Can form and speak real sentences
```

### Lesson Structure (every lesson follows this format)
1. **Hook** — One Quranic ayah or common phrase using today's concept
2. **Learn** — Core concept explained (English or Urdu based on user setting)
3. **Practice** — 3–5 exercises (flashcard / fill-blank / multiple choice)
4. **Ustadh Noor tip** — One contextual tip from the AI tutor
5. **Review words** — Vocabulary earned this lesson added to the word bank

## Language Notes for Developers

Every lesson has two fields in the content JSON:

```json
{
  "explanation_en": "The Mubtada is the topic of a nominal sentence...",
  "explanation_ur": "مبتدا اسمیہ جملے کا موضوع ہوتا ہے...",
  "arabic": "الكِتَابُ جَدِيدٌ",
  "transliteration": "Al-kitābu jadīdun",
  "meaning_en": "The book is new",
  "meaning_ur": "کتاب نئی ہے"
}
```

The app shows `explanation_en` or `explanation_ur` based on the user's language preference set during onboarding. Arabic script, transliteration, and meaning are always shown in both.
