## سبق ۱۸ — میرا مدرسہ (مَدْرَسَتِي)

### مربوط پیراگراف

| عربی | ترجمہ |
|------|--------|
| هٰذِهِ مَدْرَسَتِي | یہ میرا مدرسہ ہے |
| هِيَ قَرِيبَةٌ مِنَ الْمَسْجِدِ | یہ مسجد کے قریب ہے |
| هٰذِهِ مَدْرَسَةٌ كَبِيرَةٌ | یہ بڑا مدرسہ ہے |
| لَهَا ثَلَاثَةُ أَبْوَابٍ | اس کے تین دروازے ہیں |
| أَبْوَابُهَا مَفْتُوحَةٌ الْآنَ | اس کے دروازے ابھی کھلے ہوئے ہیں |
| فِي الْمَدْرَسَةِ فُصُولٌ كَثِيرَةٌ | مدرسے میں بہت کلاسیں ہیں |
| هٰذِهِ فَصْلِي | یہ میری کلاس ہے |
| هُوَ فَصْلٌ وَاسِعٌ | یہ ایک کشادہ کلاس ہے |
| فِيهِ نَافِذَتَانِ كَبِيرَتَانِ | اس میں دو بڑی کھڑکیاں ہیں |
| فِيهِ مَكَاتِبُ وَكَرَاسِيُّ | اس میں میزیں اور کرسیاں ہیں |
| فِيهِ سَبُّورَةٌ كَبِيرَةٌ | اس میں ایک بڑا بورڈ ہے |
| هٰذَا مَكْتَبُ الْمُعَلِّمِ | یہ استاد کی میز ہے |
| وَهٰذِهِ كُرْسِيُّهُ | اور یہ ان کی کرسی ہے |
| تِلْكَ مَكَاتِبُ الطُّلَّابِ | وہ طالبِ علموں کی میزیں ہیں |
| مَكْتَبُ الْمُعَلِّمِ كَبِيرٌ | استاد کی میز بڑی ہے |
| وَمَكَاتِبُ الطُّلَّابِ صَغِيرَةٌ | اور طالبِ علموں کی میزیں چھوٹی ہیں |
| فِي فَصْلِنَا عَشَرَةُ طُلَّابٍ | ہماری کلاس میں دس طالبِ علم ہیں |
| هُمْ مِنْ بِلَادٍ مُخْتَلِفَةٍ | وہ مختلف ممالک سے ہیں |

## رنگوں کے نام (أَلْوَان)

رنگوں کے نام **أَفْعَل** کے وزن پر آتے ہیں — یہ **غیر منصرف** ہیں:

| عربی (مذکر) | عربی (مؤنث) | رنگ |
|------------|------------|-----|
| أَحْمَرُ | حَمْرَاءُ | سرخ / لال |
| أَبْيَضُ | بَيْضَاءُ | سفید |
| أَسْوَدُ | سَوْدَاءُ | کالا |
| أَزْرَقُ | زَرْقَاءُ | نیلا |
| أَصْفَرُ | صَفْرَاءُ | پیلا |
| أَخْضَرُ | خَضْرَاءُ | سبز / ہرا |

### رنگوں کی مثالیں

| عربی | ترجمہ |
|------|--------|
| هٰذَا قَلَمٌ أَحْمَرُ | یہ ایک لال قلم ہے |
| هٰذَا قَلَمٌ أَزْرَقُ | یہ ایک نیلا قلم ہے |
| هٰذَا قَلَمٌ أَخْضَرُ | یہ ایک سبز قلم ہے |
| هٰذَا قَلَمٌ أَسْوَدُ | یہ ایک کالا قلم ہے |
| هٰذَا قَلَمٌ أَصْفَرُ | یہ ایک پیلا قلم ہے |
| هٰذِهِ مَنَادِيلُ بَيْضَاءُ | یہ سفید رومال ہیں |
| ذٰلِكَ أَصْفَرُ | وہ پیلا ہے |
| تِلْكَ حَمْرَاءُ | وہ سرخ ہے |
| هٰذِهِ زَرْقَاءُ | یہ نیلی ہے |

> **یاد رہے:** رنگ غیر منصرف ہیں — ان پر تنوین نہیں آتی

---

## غیر منصرف — مکمل اقسام

وہ اسم جن پر **تنوین نہیں آتی**:

### ۱. عورتوں کے نام
زَيْنَبُ، فَاطِمَةُ، مَرْيَمُ، خَدِيجَةُ، آمِنَةُ، سُعَادُ

### ۲. گول تا والے مردوں کے نام
طَلْحَةُ، مُعَاوِيَةُ، أُسَامَةُ، حَمْزَةُ

### ۳. الف نون پر ختم ہونے والے نام
عُثْمَانُ، سُفْيَانُ، مَرْوَانُ، نُعْمَانُ، عِمْرَانُ

### ۴. غیر عربی نام
إِبْرَاهِيمُ، إِسْمَاعِيلُ، يُوسُفُ، يَعْقُوبُ، يُونُسُ، إِسْحَاقُ

### ۵. شہروں اور ممالک کے نام
مَكَّةُ، جُدَّةُ، بَغْدَادُ، بَارِيسُ، لَنْدَنُ، لَاهُورُ

### ۶. الف نون والی صفات (مبالغہ)
كَسْلَانُ، جَوْعَانُ، عَطْشَانُ، غَضْبَانُ، مَلْآنُ، شَبْعَانُ

### ۷. أَفْعَل کے وزن پر — رنگ اور سپرلیٹو
أَحْمَرُ، أَبْيَضُ، أَسْوَدُ، أَزْرَقُ، أَصْفَرُ، أَخْضَرُ
أَكْبَرُ، أَحْسَنُ، أَجْمَلُ، أَفْضَلُ

## جمعِ مکسر — عاقل اور غیر عاقل کا فرق

| | عاقل (انسان) | غیر عاقل (بے جان/جانور) |
|--|------------|----------------------|
| اشارہ | هٰؤُلَاءِ / أُولٰئِكَ | هٰذِهِ / تِلْكَ |
| ضمیر | هُمْ | هِيَ |
| صفت | جمع | واحد مؤنث |

**مثالیں:**

| عربی | ترجمہ |
|------|--------|
| هٰؤُلَاءِ طُلَّابٌ جُدُدٌ | یہ نئے طالبِ علم ہیں (عاقل) |
| هٰذِهِ كُتُبٌ جَدِيدَةٌ | یہ نئی کتابیں ہیں (غیر عاقل) |
| تِلْكَ سَيَّارَاتٌ | وہ گاڑیاں ہیں (غیر عاقل) |
| أُولٰئِكَ مُعَلِّمُونَ | وہ استاد ہیں (عاقل) |

## ہوم ورک

- رنگوں کے نام مذکر اور مؤنث دونوں میں یاد کریں
- اپنے گھر / کلاس کی چیزوں کے رنگ عربی میں بتائیں
- غیر منصرف کی ساتوں اقسام دہرائیں
- سبق "مَدْرَسَتِي" کا پیراگراف خود لکھیں
