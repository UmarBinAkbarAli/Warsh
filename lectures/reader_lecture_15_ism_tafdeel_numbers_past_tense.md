# ریڈر لیکچر ۴ — اسمِ تفضیل، اعداد، اور فعلِ ماضی

بِسْمِ اللهِ الرَّحْمٰنِ الرَّحِيْم

اس لیکچر میں چار بڑے موضوعات ہیں — تینوں گرامر کے بنیادی ستون، اور پھر ریڈر کا سبق ۴ جس میں ہم پہلی بار **فعلِ ماضی** کا مکمل استعمال دیکھیں گے۔

---

## حصہ ۱: اسمِ تفضیل (Comparative & Superlative)

### اسمِ تفضیل کیا ہے؟

**اسمِ تفضیل** = وہ اسم جو دو یا زیادہ چیزوں کے درمیان موازنہ ظاہر کرے۔

- اردو میں: **زیادہ، سے بڑھ کر، سب سے ...**
- انگریزی میں: **-er / -est** (taller, tallest)

### بنیادی وزن: **أَفْعَل**

| اصل صفت | اسمِ تفضیل | معنی |
|---------|------------|------|
| كَبِير (بڑا) | أَكْبَر | زیادہ بڑا / سب سے بڑا |
| صَغِير (چھوٹا) | أَصْغَر | زیادہ چھوٹا / سب سے چھوٹا |
| طَوِيل (لمبا) | أَطْوَل | زیادہ لمبا |
| قَصِير (چھوٹا/پستہ) | أَقْصَر | زیادہ چھوٹا |
| جَمِيل (خوبصورت) | أَجْمَل | زیادہ خوبصورت |
| سَهْل (آسان) | أَسْهَل | زیادہ آسان |
| قَدِيم (پرانا) | أَقْدَم | زیادہ پرانا |
| رَخِيص (سستا) | أَرْخَص | زیادہ سستا |
| نَظِيف (صاف) | أَنْظَف | زیادہ صاف |
| بَعِيد (دور) | أَبْعَد | زیادہ دور |
| قَرِيب (قریب) | أَقْرَب | زیادہ قریب |

---

### استعمال کا قاعدہ — Comparative (تفضیلی)

**ساخت:**

> **[اسمِ تفضیل] + مِن + [مقابلہ والی چیز]**

یعنی **مِنْ** کا ترجمہ یہاں "سے" ہوگا۔

### مثالیں:

| عربی جملہ | اردو ترجمہ |
|-----------|------------|
| هٰذَا الكِتَابُ أَسْهَلُ مِنْ ذٰلِكَ | یہ کتاب اُس سے زیادہ آسان ہے |
| هٰذِهِ السَّيَّارَةُ أَجْمَلُ مِنْ تِلْكَ | یہ گاڑی اُس سے زیادہ خوبصورت ہے |
| حَمْزَةُ أَكْبَرُ مِنِّي سِنًّا | حمزہ عمر میں مجھ سے بڑا ہے |
| بَيْتِي أَبْعَدُ مِنْ بَيْتِكَ | میرا گھر تمہارے گھر سے زیادہ دور ہے |
| اللَّبَنُ أَحْسَنُ مِنَ الشَّايِ | دودھ چائے سے بہتر ہے |
| الكَاهِرَةُ أَكْبَرُ مِنَ الرِّيَاضِ | قاہرہ ریاض سے بڑا ہے |
| الرِّجَالُ أَكْثَرُ مِنَ النِّسَاءِ | مرد عورتوں سے زیادہ ہیں |

> **اہم نکتہ:** اسمِ تفضیل **ہمیشہ مذکر مفرد** کی شکل میں ہی آتا ہے، چاہے مبتدا مؤنث ہو یا جمع۔
> مثلاً: السَّيَّارَةُ أَجْمَلُ (نہ کہ "جَمْلَى")

---

### استعمال کا قاعدہ — Superlative (سب سے ...)

**ساخت:**

> **اَلْ + [اسمِ تفضیل] + اضافت کی شکل**

یا

> **[اسمِ تفضیل] + اسمِ نکرہ مجرور**

### مثالیں:

| عربی جملہ | اردو ترجمہ |
|-----------|------------|
| هٰذَا أَحْسَنُ بَيْتٍ فِي الشَّارِعِ | یہ گلی کا سب سے اچھا گھر ہے |
| آمِنَةُ أَصْغَرُ طَالِبَةٍ فِي المَدْرَسَةِ | آمنہ مدرسے کی سب سے چھوٹی طالبہ ہے |
| بِلَالٌ أَحْسَنُ لاَعِبٍ فِي فَرِيقِنَا | بلال ہماری ٹیم کا سب سے اچھا کھلاڑی ہے |
| يَاسِرٌ أَفْقَرُ رَجُلٍ فِي قَرْيَتِنَا | یاسر ہمارے گاؤں کا سب سے غریب آدمی ہے |
| الدُّكْتُور زَكَرِيَّا أَشْهَرُ طَبِيبٍ فِي بَلَدِنَا | ڈاکٹر زکریا ہمارے ملک کے سب سے مشہور ڈاکٹر ہیں |
| الأَزْهَرُ أَقْدَمُ جَامِعَةٍ فِي العَالَمِ | الازہر دنیا کی سب سے پرانی یونیورسٹی ہے |

> **یاد رکھیں:** Superlative میں دوسرا اسم **نکرہ مجرور** (تنوینِ زیر) آتا ہے۔

## حصہ ۲: لٰكِنَّ اور كَأَنَّ (إنّ کے بھائی)

یہ دونوں حروف **حروفِ مشبہ بالفعل** میں سے ہیں — یعنی یہ **إِنَّ** کی طرح کام کرتے ہیں۔

### قاعدہ:

> **لٰكِنَّ / كَأَنَّ کے بعد اسم آئے تو وہ نصب میں (زبر کے ساتھ) ہوگا۔**

### لٰكِنَّ — "لیکن"

| عربی جملہ | اردو ترجمہ |
|-----------|------------|
| الطُّلَّابُ كَثِيرُونَ، لٰكِنَّ الفُصُولَ صَغِيرَةٌ | طلباء بہت ہیں، لیکن کلاسیں چھوٹی ہیں |
| حَامِدٌ مُجْتَهِدٌ، لٰكِنَّ صَدِيقَهُ كَسْلَانُ | حامد محنتی ہے، لیکن اُس کا دوست سست ہے |
| هٰذَا الدَّرْسُ طَوِيلٌ، لٰكِنَّهُ سَهْلٌ | یہ سبق لمبا ہے، لیکن (وہ) آسان ہے |
| أَخِي مُتَزَوِّجٌ، لٰكِنَّنِي عَازِبٌ | میرا بھائی شادی شدہ ہے، لیکن میں کنوارا ہوں |
| إِبْرَاهِيمُ فَقِيرٌ، لٰكِنَّ عَمَّهُ غَنِيٌّ جِدًّا | ابراہیم غریب ہے، لیکن اس کا چچا بہت امیر ہے |

### كَأَنَّ — "گویا کہ / جیسے"

| عربی جملہ | اردو ترجمہ |
|-----------|------------|
| مَنْ هٰذَا الفَتَى؟ كَأَنَّهُ أَخُوكَ | یہ نوجوان کون ہے؟ گویا یہ آپ کا بھائی ہے |
| مَنْ هٰذِهِ الطِّفْلَةُ؟ كَأَنَّهَا أُخْتُكَ | یہ بچی کون ہے؟ گویا یہ آپ کی بہن ہے |
| كَأَنَّكَ مَرِيضٌ | گویا تم بیمار ہو |
| كَأَنَّكَ أَخِي | گویا تم میرے بھائی (جیسے) ہو |

## حصہ ۳: اعداد — ۱۱ سے ۲۰ تک

### بنیادی فہرست

| عدد | مذکر | مؤنث |
|-----|------|------|
| ۱۱ | أَحَدَ عَشَرَ | إِحْدَى عَشْرَةَ |
| ۱۲ | اِثْنَا عَشَرَ | اِثْنَتَا عَشْرَةَ |
| ۱۳ | ثَلَاثَةَ عَشَرَ | ثَلَاثَ عَشْرَةَ |
| ۱۴ | أَرْبَعَةَ عَشَرَ | أَرْبَعَ عَشْرَةَ |
| ۱۵ | خَمْسَةَ عَشَرَ | خَمْسَ عَشْرَةَ |
| ۱۶ | سِتَّةَ عَشَرَ | سِتَّ عَشْرَةَ |
| ۱۷ | سَبْعَةَ عَشَرَ | سَبْعَ عَشْرَةَ |
| ۱۸ | ثَمَانِيَةَ عَشَرَ | ثَمَانِيَ عَشْرَةَ |
| ۱۹ | تِسْعَةَ عَشَرَ | تِسْعَ عَشْرَةَ |
| ۲۰ | عِشْرُونَ | عِشْرُونَ |

### قواعد:

1. **۱۱ سے ۱۹ تک تمام اعداد مبنی علی الفتح ہیں** — یعنی ہمیشہ زبر کے ساتھ آتے ہیں، چاہے رفع، نصب، جر کوئی بھی حالت ہو۔

2. **عدد ۱۲ مستثنیٰ ہے** — اس کا پہلا حصہ (اِثْنَا) رفع میں الف کے ساتھ، اور نصب و جر میں **اِثْنَيْ** ہوجاتا ہے (نون گرتا ہے اضافت کی وجہ سے)۔

3. **مذکر/مؤنث کا قاعدہ:**
   - عدد ۱۱ اور ۱۲ میں: عدد کے **دونوں حصے** معدود کے ساتھ موافق ہوں گے (sama jender)
   - عدد ۱۳ سے ۱۹ تک: عدد کا **پہلا حصہ** معدود کے **مخالف** جنس کا ہوگا، اور **دوسرا حصہ** معدود کے موافق ہوگا

4. **معدود ہمیشہ مفرد منصوب (تمیز) آتا ہے** — یعنی واحد، فتح کے ساتھ، اور تنوین کے ساتھ۔

### مثالیں:

| عربی جملہ | اردو ترجمہ |
|-----------|------------|
| فِي الفَصْلِ تِسْعَةَ عَشَرَ طَالِبًا | کلاس میں ۱۹ طالب علم ہیں |
| عِنْدِي خَمْسَ عَشْرَةَ كِتَابًا | میرے پاس ۱۵ کتابیں ہیں |
| رَأَى يُوسُفُ عَلَيْهِ السَّلَامُ فِي المَنَامِ أَحَدَ عَشَرَ كَوْكَبًا | یوسف علیہ السلام نے خواب میں ۱۱ ستارے دیکھے |
| مَعِي عِشْرُونَ رِيَالًا | میرے پاس ۲۰ ریال ہیں |
| فِي هٰذَا الشَّارِعِ ثَلَاثَةَ عَشَرَ فُنْدُقًا | اس سڑک پر ۱۳ ہوٹل ہیں |
| فِي هٰذِهِ المَدِينَةِ أَرْبَعَ عَشْرَةَ مَسْجِدًا | اس شہر میں ۱۴ مسجدیں ہیں |
| ثَمَنُ هٰذَا الكِتَابِ سَبْعَةَ عَشَرَ رِيَالًا | اس کتاب کی قیمت ۱۷ ریال ہے |
| فِي هٰذَا الكِتَابِ عِشْرُونَ دَرْسًا | اس کتاب میں ۲۰ سبق ہیں |
| فِي السَّنَةِ اِثْنَا عَشَرَ شَهْرًا | سال میں ۱۲ مہینے ہیں |

## حصہ ۴: اعدادِ ترتیبی (Ordinal Numbers)

یہ اعداد **ترتیب** بتاتے ہیں — پہلا، دوسرا، تیسرا...

### فہرست (۱ سے ۱۰ تک):

| نمبر | عدد ترتیبی (مذکر) | معنی |
|------|--------------------|------|
| ۱ | اَلْأَوَّل | پہلا |
| ۲ | اَلثَّانِي | دوسرا |
| ۳ | اَلثَّالِث | تیسرا |
| ۴ | اَلرَّابِع | چوتھا |
| ۵ | اَلْخَامِس | پانچواں |
| ۶ | اَلسَّادِس | چھٹا |
| ۷ | اَلسَّابِع | ساتواں |
| ۸ | اَلثَّامِن | آٹھواں |
| ۹ | اَلتَّاسِع | نواں |
| ۱۰ | اَلْعَاشِر | دسواں |

> **نکتہ:** ۳ سے ۱۰ تک کے اعدادِ ترتیبی **اسمِ فاعل** کے وزن (فَاعِل) پر آتے ہیں۔ ۱، ۲ مستثنیٰ ہیں۔

### استعمال:

اعدادِ ترتیبی **صفت** کی طرح آتے ہیں — یعنی موصوف کے ساتھ:
- **جنس** میں موافقت
- **عدد** میں موافقت (مفرد/جمع)
- **اعراب** میں موافقت

### مثالیں:

| عربی | اردو |
|------|------|
| اَلدَّرْسُ الْأَوَّل | پہلا سبق |
| اَلدَّرْسُ الثَّانِي | دوسرا سبق |
| اَلدَّرْسُ الثَّالِث | تیسرا سبق |
| اَلْيَوْمُ الثَّامِن | آٹھواں دن |
| اَلسَّنَةُ الرَّابِعَة | چوتھا سال (مؤنث: رَابِعَة) |
| فِي السَّنَةِ الرَّابِعَةِ | چوتھے سال میں |
| اَلْبَيْتُ السَّابِع | ساتواں گھر |
| اَلصَّفْحَةُ التَّاسِعَة | نواں صفحہ |
| اَلْغُرْفَةُ الْخَامِسَة | پانچواں کمرہ |

> **یاد رکھیں:** "اَلْأَوَّل" کا مؤنث **اَلْأُولَى** ہے، اور "اَلثَّانِي" کا مؤنث **اَلثَّانِيَة** ہے۔

## حصہ ۵: ریڈر سبق ۴ — فعلِ ماضی کی مشق

اب ہم بات چیت کی مشق دیکھتے ہیں جس میں **ذَهَبَ (گیا)** اور **رَجَعَ (لوٹا)** کا کثرت سے استعمال ہے۔

### مکالمہ: یوسف اور ابراہیم

| عربی | اردو ترجمہ |
|------|------------|
| السَّلَامُ عَلَيْكُمْ وَرَحْمَةُ اللهِ وَبَرَكَاتُهُ | سلام علیکم |
| وَعَلَيْكُمُ السَّلَامُ وَرَحْمَةُ اللهِ وَبَرَكَاتُهُ، أَهْلًا وَسَهْلًا، مَرْحَبًا | و علیکم السلام... خوش آمدید |
| كَيْفَ حَالُكَ يَا خَالِي؟ | میرے ماموں! آپ کیسے ہیں؟ |
| الحَمْدُ لِلهِ، أَيْنَ أَبُوكَ؟ | الحمد للہ، آپ کے ابو کہاں ہیں؟ |
| ذَهَبَ إِلَى السُّوقِ | وہ بازار گئے ہیں |
| وَأَيْنَ أُمُّكَ؟ | اور آپ کی امی کہاں ہیں؟ |
| ذَهَبَتْ إِلَى خَالَتِي زَيْنَب | وہ میری خالہ زینب کی طرف گئی ہیں |
| وَأَيْنَ إِخْوَتُكَ؟ | اور آپ کے بھائی کہاں ہیں؟ |
| ذَهَبُوا إِلَى الجَامِعَةِ | وہ یونیورسٹی گئے ہیں |
| وَأَيْنَ أَخَوَاتُكَ؟ | اور آپ کی بہنیں کہاں ہیں؟ |
| ذَهَبْنَ إِلَى المَدْرَسَةِ | وہ مدرسے گئی ہیں |

### فعلِ ماضی کی گردان (ذَهَبَ):

| ضمیر | فعل | معنی |
|-------|------|------|
| هُوَ | ذَهَبَ | وہ (مذکر) گیا |
| هِيَ | ذَهَبَتْ | وہ (مؤنث) گئی |
| هُمْ | ذَهَبُوا | وہ سب (مذکر) گئے |
| هُنَّ | ذَهَبْنَ | وہ سب (مؤنث) گئیں |
| أَنْتَ | ذَهَبْتَ | تو (مذکر) گیا |
| أَنْتِ | ذَهَبْتِ | تو (مؤنث) گئی |
| أَنَا | ذَهَبْتُ | میں گیا/گئی |
| نَحْنُ | ذَهَبْنَا | ہم گئے |

> **اہم نکتہ:** **نَحْنُ** اور **هُنَّ** کی فعل ایک جیسی ہے (ذَهَبْنَ / ذَهَبْنَا میں صرف "ا" کا فرق)۔

### سوالیہ کلمات اس مکالمے میں:

| کلمہ | معنی |
|------|------|
| كَيْفَ | کیسے |
| أَيْنَ | کہاں |
| أَمَا | کیا نہیں؟ (نفی کے ساتھ سوال) |
| لِمَاذَا | کیوں |
| مَتَى | کب |
| هَلْ | کیا (سادہ سوال) |
| أَلَا | کیا نہیں؟ |

### مثالیں:

| عربی | اردو |
|------|------|
| أَمَا ذَهَبْتَ إِلَى المَدْرَسَةِ؟ | کیا تم مدرسے نہیں گئے؟ |
| لِمَاذَا رَجَعْتَ؟ | تم واپس کیوں آئے؟ |
| رَجَعْتُ لِأَنِّي مَرِيضٌ | میں واپس آیا کیونکہ میں بیمار ہوں |
| مَتَى رَجَعَ إِبْرَاهِيمُ مِنَ المَدْرَسَةِ؟ | ابراہیم مدرسے سے کب لوٹا؟ |
| أَذَهَبْتَ إِلَى الطَّبِيبِ؟ | کیا تم ڈاکٹر کے پاس گئے؟ |
| لَا بَأْسَ | کوئی بات نہیں / فکر مت کرو |

## یاد رکھنے کے کلیدی نکات

1. **اسمِ تفضیل کا وزن** ہمیشہ **أَفْعَل** ہے، اور یہ ہمیشہ مذکر مفرد رہتا ہے۔
2. **Comparative** کے لیے **مِنْ** آتا ہے، **Superlative** کے لیے **اضافت** یا **اَلْ + مجرور**۔
3. **اعداد ۱۱-۱۹** میں عدد **فتح** پر مبنی ہے، اور معدود **مفرد منصوب** (تنوینِ زبر) آتا ہے۔
4. **عدد ۱۲** مستثنیٰ ہے — اس کا اعراب بدلتا ہے (نون گرتا ہے اضافت میں)۔
5. **اعدادِ ترتیبی** صفت کی طرح موصوف سے ہر طرح موافقت کرتے ہیں (جنس، عدد، اعراب)۔
6. **لٰكِنَّ اور كَأَنَّ** بھی إِنَّ کی طرح اپنے اسم کو **نصب** دیتے ہیں۔
7. **فعلِ ماضی** میں جنس اور عدد دونوں کے لحاظ سے صیغے بدلتے ہیں — کل ۱۴ صیغے ہیں۔

## اگلا سبق

اگلے سبق میں ہم **مزید افعالِ ماضی** اور **حالیہ ضمائر کے ساتھ ان کا استعمال** دیکھیں گے، نیز **مَتَى** اور **لِمَاذَا** کے مزید استعمالات۔

وَاللهُ أَعْلَم
