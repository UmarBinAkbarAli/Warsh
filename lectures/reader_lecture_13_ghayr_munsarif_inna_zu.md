## غیر منصرف — جر میں فتحہ آتا ہے

غیر منصرف اسم کی جب **حالتِ جر** ہو تو کسرہ کی جگہ **فتحہ** آتا ہے:

### قاعدہ

| قسم | رَفْع | نَصْب | جَر |
|-----|------|------|-----|
| مُنْصَرِف (مثلاً مُحَمَّدٌ) | مُحَمَّدٌ | مُحَمَّدًا | مُحَمَّدٍ (کسرہ) |
| غیر مُنْصَرِف (مثلاً إِبْرَاهِيمُ) | إِبْرَاهِيمُ | إِبْرَاهِيمَ | إِبْرَاهِيمَ (فتحہ) |

> **یاد رہے:** غیر منصرف میں نصب اور جر دونوں ایک جیسے (فتحہ کے ساتھ)

---

### غیر منصرف اسماء جر میں — مثالیں

| عربی | ترجمہ | نوٹ |
|------|--------|-----|
| ذَهَبَ إِلَى مَكَّةَ | وہ مکہ گیا | مَكَّةُ غیر منصرف — جر میں فتحہ |
| هُوَ فِي لَنْدَنَ | وہ لندن میں ہے | لَنْدَنُ غیر منصرف |
| ذَهَبَ إِلَى بَغْدَادَ | وہ بغداد گیا | بَغْدَادُ غیر منصرف |
| هِيَ لِخَدِيجَةَ | یہ خدیجہ کی ہے | خَدِيجَةُ غیر منصرف |
| جَاءَ مِنْ يُوسُفَ | یوسف کی طرف سے آیا | يُوسُفُ غیر منصرف |
| كِتَابُ إِبْرَاهِيمَ | ابراہیم کی کتاب | إِبْرَاهِيمُ جر میں — فتحہ |
| ذَهَبَ أَبِي إِلَى مَكَّةَ | میرے ابا مکہ گئے | — |
| ذَهَبَ عَمِّي إِلَى جُدَّةَ | میرے چچا جدہ گئے | — |
| زَوْجُ خَدِيجَةَ فِي لَنْدَنَ | خدیجہ کا شوہر لندن میں ہے | — |
| سَيَّارَةُ هَامِدٍ جَدِيدَةٌ | حامد کی گاڑی نئی ہے | هَامِدٌ مُنْصَرِف — کسرہ |
| سَيَّارَةُ إِبْرَاهِيمَ قَدِيمَةٌ | ابراہیم کی گاڑی پرانی ہے | إِبْرَاهِيمُ غیر منصرف — فتحہ |

### مُنْصَرِف بمقابلہ غیر مُنْصَرِف — ایک جدول میں

| | مُنْصَرِف | غیر مُنْصَرِف |
|--|---------|------------|
| مثال | مُحَمَّدٌ | زَيْنَبُ / إِبْرَاهِيمُ / مَكَّةُ |
| رفع | مُحَمَّدٌ | زَيْنَبُ |
| نصب | مُحَمَّدًا | زَيْنَبَ |
| جر | مُحَمَّدٍ (کسرہ) | زَيْنَبَ (فتحہ) |
| جمع مکسر | مَسَاجِدُ | مَسَاجِدَ (جر میں) |

> **جمعِ مکسر جو غیر منصرف ہو:** جر میں بھی فتحہ — مثلاً فِي مَسَاجِدَ، مِنْ مَدَارِسَ

| إِنَّهَا رَخِيصَةٌ جِدًّا | یہ بہت سستی ہے |
| إِنَّهَا بِمِائَةِ رُوبِيَةٍ فَقَطْ | یہ صرف سو روپے میں ہے |

| لَا، إِنَّهُمْ تُجَّارٌ | نہیں، وہ تاجر ہیں |

| أَهُنَّ فِي الْهِنْدِ الْآنَ؟ | کیا وہ ابھی ہندوستان میں ہیں؟ |
| لَا، إِنَّهُنَّ هُنَا فِي الْمَدِينَةِ الْمُنَوَّرَةِ | نہیں، وہ یہاں مدینہ منورہ میں ہیں |

## إِنَّ اور اس کی بہنیں (إِنَّ وَأَخَوَاتُهَا)

### تعارف

**إِنَّ** جملہ اسمیہ پر داخل ہوتی ہے اور دو کام کرتی ہے:
1. معنی میں تاکید لاتی ہے — **بے شک / یقیناً**
2. مُبْتَدَا کو **نصب** میں لے جاتی ہے

### إِنَّ کا قاعدہ

| جملہ بغیر إِنَّ | جملہ إِنَّ کے ساتھ | ترجمہ |
|---------------|-----------------|-------|
| هُوَ تَاجِرٌ | إِنَّهُ تَاجِرٌ | بے شک وہ تاجر ہے |
| هِيَ جَمِيلَةٌ | إِنَّهَا جَمِيلَةٌ | بے شک وہ خوبصورت ہے |
| الْقُرْآنُ كِتَابُ اللهِ | إِنَّ الْقُرْآنَ كِتَابُ اللهِ | بے شک قرآن اللہ کی کتاب ہے |

> **تبدیلی:** مُبْتَدَا (رفع میں) → إِنَّ کے بعد نصب میں آتا ہے
> اسے **اِسمُ إِنَّ** کہتے ہیں
> اور پرانی خبر کو **خبرُ إِنَّ** کہتے ہیں — خبر رفع میں رہتی ہے

---

### إِنَّ کے ساتھ ضمائر کی تبدیلی

جب ضمیر إِنَّ کے بعد آئے تو منفصل ضمیر (رفع) کی جگہ متصل ضمیر (نصب/جر والی) آتی ہے:

| منفصل ضمیر | إِنَّ کے ساتھ | ترجمہ |
|----------|------------|-------|
| أَنَا | إِنِّي / إِنَّنِي | بے شک میں |
| أَنْتَ | إِنَّكَ | بے شک تم |
| أَنْتِ | إِنَّكِ | بے شک تم (مؤنث) |
| هُوَ | إِنَّهُ | بے شک وہ |
| هِيَ | إِنَّهَا | بے شک وہ (مؤنث) |
| نَحْنُ | إِنَّنَا | بے شک ہم |
| أَنْتُمْ | إِنَّكُمْ | بے شک تم سب |
| أَنْتُنَّ | إِنَّكُنَّ | بے شک تم سب (مؤنث) |
| هُمْ | إِنَّهُمْ | بے شک وہ (جمع مذکر) |
| هُنَّ | إِنَّهُنَّ | بے شک وہ (جمع مؤنث) |

## لَعَلَّ — شاید (إِنَّ کی بہن)

**لَعَلَّ** = شاید | یہ بھی إِنَّ کی طرح اسم کو نصب دیتی ہے:

| عربی | ترجمہ |
|------|--------|
| لَعَلَّهُ بِخَيْرٍ | شاید وہ خیریت سے ہے |
| لَعَلَّهُمْ بِخَيْرٍ | شاید وہ سب خیریت سے ہیں |
| لَعَلَّهَا بِخَيْرٍ | شاید وہ (عورت) خیریت سے ہے |
| لَعَلَّهُنَّ بِخَيْرٍ | شاید وہ عورتیں خیریت سے ہیں |
| لَعَلَّكَ بِخَيْرٍ | شاید تم خیریت سے ہو |
| لَعَلَّكُمْ بِخَيْرٍ | شاید تم سب خیریت سے ہو |
| لَعَلَّكِ بِخَيْرٍ | شاید تم (مؤنث) خیریت سے ہو |
| لَعَلَّكُنَّ بِخَيْرٍ | شاید تم سب (مؤنث) خیریت سے ہو |
| لَعَلَّنِي نَاجِحٌ | شاید میں کامیاب ہوں |
| لَعَلَّنَا نَاجِحُونَ | شاید ہم کامیاب ہیں |
| لَعَلَّ الْمُدِيرَ فِي غُرْفَتِهِ | شاید پرنسپل اپنے کمرے میں ہیں |
| لَعَلَّ الِامْتِحَانَ سَهْلٌ | شاید امتحان آسان ہے |

## یا یا کا استعمال (أَمْ)

**أَمْ** = یا (سوال میں دو آپشن دینے کے لیے):

| عربی | ترجمہ |
|------|--------|
| أَمِنَ الْبَاكِسْتَانِ أَنْتَ أَمْ مِنَ الْهِنْدِ؟ | کیا آپ پاکستان سے ہیں یا ہند سے؟ |
| أَطَبِيبٌ أَنْتَ أَمْ مُهَنْدِسٌ؟ | کیا آپ ڈاکٹر ہیں یا انجینئر؟ |
| أَهُوَ مِنَ الصِّينِ أَمْ مِنَ الْيَابَانِ؟ | کیا وہ چین سے ہے یا جاپان سے؟ |
| أَهِيَ غَالِيَةٌ أَمْ رَخِيصَةٌ؟ | کیا یہ مہنگی ہے یا سستی؟ |
| أَمُتَزَوِّجٌ أَنْتَ أَمْ أَعْزَبُ؟ | کیا آپ شادی شدہ ہیں یا کنوارے؟ |
| أَمَسْجِدٌ هٰذَا أَمْ مَدْرَسَةٌ؟ | کیا یہ مسجد ہے یا مدرسہ؟ |

> **نوٹ:** خبر کو پہلے لایا جا سکتا ہے جب سوال میں choice ہو

---

## ذُو / ذَاتُ — والا / والی

**ذُو** (مذکر) = والا | **ذَاتُ** (مؤنث) = والی

یہ **مُضَاف** ہیں — ان کے بعد اسم جر میں آتا ہے:

| | مذکر واحد | مؤنث واحد | جمع مذکر | جمع مؤنث |
|--|----------|----------|---------|---------|
| والا/والی | ذُو | ذَاتُ | ذَوُو | ذَوَاتُ |

### ذُو / ذَاتُ کی مثالیں

| عربی | ترجمہ |
|------|--------|
| مُدِيرُنَا ذُو لِحْيَةٍ طَوِيلَةٍ | ہمارے ڈائریکٹر لمبی داڑھی والے ہیں |
| هٰذَا دَفْتَرٌ ذُو سُطُورٍ | یہ لکیروں والا رجسٹر ہے |
| أَحْمَدُ طَالِبٌ ذُو عِلْمٍ وَأَخْلَاقٍ | احمد علم اور اخلاق والا طالبِ علم ہے |
| هٰذَا الْقَمِيصُ ذُو كُمٍّ قَصِيرٍ | یہ کم آستین والی قمیص ہے |
| وَذَاكَ ذُو كُمٍّ طَوِيلٍ | اور وہ لمبی آستین والی ہے |
| الْمَسْجِدُ ذُو مِئْذَنَةٍ وَاحِدَةٍ | مسجد ایک مینار والی ہے |
| هٰذِهِ الْمَرْأَةُ ذَاتُ مَالٍ كَثِيرٍ | یہ عورت بہت مال والی ہے |
| رَجُلٌ ذُو مَالٍ كَثِيرٍ | بہت مال والا آدمی |

### ذُو کی حالتیں

| حالت | شکل | مثال |
|------|-----|-------|
| رَفْع | ذُو | هُوَ ذُو مَالٍ |
| نَصْب | ذَا | رَأَيْتُ رَجُلًا ذَا مَالٍ |
| جَر | ذِي | مَرَرْتُ بِرَجُلٍ ذِي مَالٍ |
| جمع | ذَوُو / ذَوِي | هُمْ ذَوُو عِلْمٍ |
| مؤنث | ذَاتُ | هِيَ ذَاتُ أَخْلَاقٍ |
| مؤنث جمع | ذَوَاتُ | هُنَّ ذَوَاتُ عِلْمٍ |

## قیمتیں (أَسْعَار)

سبق میں قیمتوں کا بھی تذکرہ ہوا — **بِ** (میں / کے بدلے) حرفِ جار ہے:

| عربی | ترجمہ |
|------|--------|
| بِكَمْ هٰذَا؟ | یہ کتنے میں ہے؟ |
| إِنَّهَا بِمِائَةِ رُوبِيَةٍ فَقَطْ | یہ صرف سو روپے میں ہے |
| هٰذِهِ بِأَلْفِ رِيَالٍ | یہ ہزار ریال میں ہے |
| إِنَّهُ بِمِائَةِ رِيَالٍ | یہ سو ریال میں ہے |
| عِنْدِي أَلْفُ دُولَارٍ | میرے پاس ایک ہزار ڈالر ہے |
| هٰذَا الْمُعْجَمُ بِمِائَةِ رِيَالٍ | یہ ڈکشنری سو ریال کی ہے |
| هٰذِهِ الْحَقِيبَةُ غَالِيَةٌ | یہ بیگ مہنگا ہے |
| اَلسَّاعَةُ رَخِيصَةٌ جِدًّا | گھڑی بہت سستی ہے |

### گنتی کے ساتھ قیمت — مُرَكَّبِ اِضَافِي

| عربی | ترجمہ | نوٹ |
|------|--------|-----|
| مِائَةُ رُوبِيَةٍ | سو روپے | مِائَةٌ مُضَاف |
| أَلْفُ رِيَالٍ | ہزار ریال | أَلْفٌ مُضَاف |
| مِائَةُ رَجُلٍ | سو آدمی | — |
| أَلْفُ امْرَأَةٍ | ہزار عورتیں | — |

### مشق الف — إِنَّ داخل کریں

### مشق ب — غیر منصرف جر میں

| اسم | حرفِ جار + اسم |
|-----|-------------|
| إِبْرَاهِيمُ | إِلَى إِبْرَاهِيمَ |

### مشق ج — ذُو / ذَاتُ لگائیں

| جملہ | ذُو / ذَاتُ کے ساتھ |
|------|------------------|
| وہ لمبی داڑھی والا ہے | هُوَ ذُو لِحْيَةٍ طَوِيلَةٍ |

| یہ طالبِ علم اخلاق والا ہے | هُوَ طَالِبٌ ذُو أَخْلَاقٍ |

## ہوم ورک

- إِنَّ والے ۱۰ جملے بنائیں (ضمائر سمیت)
- لَعَلَّ والے ۵ جملے بنائیں
- غیر منصرف ناموں کو جر میں لکھنے کی مشق کریں
- ذُو / ذَاتُ کے ساتھ ۵ جملے بنائیں
- بِكَمْ — قیمت پوچھنے اور بتانے کے ۵ جملے

## خلاصہ

| موضوع | اہم نکتہ |
|-------|----------|
| غیر منصرف جر میں | کسرہ کی جگہ فتحہ — مُنْصَرِف میں کسرہ |
| إِنَّ کا کام | مُبْتَدَا → نصب + معنی میں تاکید (بے شک) |
| إِنَّ + ضمیر | منفصل → متصل شکل |
| لَعَلَّ | شاید — وہی قاعدہ جو إِنَّ کا |
| أَمْ | یا — choice والے سوال میں |
| ذُو / ذَاتُ | والا / والی — مُضَاف، بعد میں جر |
| قیمت | بِكَمْ — مُرَكَّبِ اِضَافِي |
