# ریڈر لیکچر ۱۰ — سبق ۱۴ اور ۱۵: ضمائر جمع اور مکالمہ

## بسم اللہ الرحمٰن الرحیم

| ضمیر | مطلب | قسم |
|------|------|-----|
| أَنَا | میں | متکلم واحد |
| نَحْنُ | ہم | متکلم جمع |
| أَنْتَ | تم | مخاطب واحد مذکر |

## متصل ضمائر جمع — اسم کے ساتھ

| ضمیر | متصل | مثال | ترجمہ |
|------|------|-------|--------|
| نَا (ہمارا) | ـنَا | بَيْتُنَا | ہمارا گھر |
| كُمْ (تم سب کا مذکر) | ـكُمْ | بَيْتُكُمْ | تم سب کا گھر |
| كُنَّ (تم سب کا مؤنث) | ـكُنَّ | بَيْتُكُنَّ | تم سب کا گھر |
| هُمْ (ان کا مذکر) | ـهُمْ | بَيْتُهُمْ | ان کا گھر |
| هُنَّ (ان کا مؤنث) | ـهُنَّ | بَيْتُهُنَّ | ان کا گھر |

### مشق جدول — ایک اسم، سب ضمائر

| | واحد | جمع |
|--|------|-----|
| میرا | أُمِّي | أُمُّنَا |
| تیرا (مذکر) | أُمُّكَ | أُمُّكُمْ |
| تیری (مؤنث) | أُمُّكِ | أُمُّكُنَّ |
| اس کا (مذکر) | أُمُّهُ | أُمُّهُمْ |
| اس کی (مؤنث) | أُمُّهَا | أُمُّهُنَّ |

---

## فعل کی جمع شکلیں

| واحد مذکر | واحد مؤنث | جمع مذکر | جمع مؤنث |
|---------|---------|---------|---------|
| ذَهَبَ | ذَهَبَتْ | ذَهَبُوا | ذَهَبْنَ |
| خَرَجَ | خَرَجَتْ | خَرَجُوا | خَرَجْنَ |
| جَلَسَ | جَلَسَتْ | جَلَسُوا | جَلَسْنَ |

> **قاعدہ:** جمع مذکر = فعل + **وا** | جمع مؤنث = فعل + **نَ**

---

## قَبْلَ اور بَعْدَ

**قَبْلَ** = پہلے | **بَعْدَ** = بعد

دونوں **مُضَاف** ہیں — ان کے بعد اسم **جر** میں آتا ہے:

| عربی | ترجمہ |
|------|--------|
| قَبْلَ الصَّلَاةِ | نماز سے پہلے |
| بَعْدَ الصَّلَاةِ | نماز کے بعد |
| قَبْلَ أُسْبُوعٍ | ایک ہفتہ پہلے |
| بَعْدَ شَهْرٍ | ایک مہینے بعد |
| قَبْلَ ثَلَاثَةِ أَيَّامٍ | تین دن پہلے |

## سبق ۱۵ — مکالمہ

### ڈاکٹر کے گھر کا دورہ

| عربی | ترجمہ |
|------|--------|
| اَلسَّلَامُ عَلَيْكُمْ | السلام علیکم |
| مَرْحَبًا! أَبُوكُمْ صَدِيقِي | خوش آمدید! تمہارے ابا میرے دوست ہیں |
| أَيْنَ عَمُّكَ عِيسَى؟ | تمہارے چچا عیسیٰ کہاں ہیں؟ |
| هُوَ مَرِيضٌ، هُوَ الْآنَ فِي الْمُسْتَشْفَى | وہ بیمار ہیں، ابھی ہسپتال میں ہیں |
| شِفَاءٌ وَعَافِيَةٌ | شفا اور عافیت ہو |
| مَنِ الطِّفْلَةُ الَّتِي مَعَكَ؟ | تمہارے ساتھ یہ بچی کون ہے؟ |
| هِيَ أُخْتِي | وہ میری بہن ہے |
| بَيْتُنَا الْجَدِيدُ أَمَامَنَا | ہمارا نیا گھر آگے بنا ہوا ہے |

## اہم جملے — ہم سب

| عربی | ترجمہ |
|------|--------|
| نَحْنُ كُلُّنَا مُسْلِمُونَ | ہم سب مسلمان ہیں |
| دِينُنَا الإِسْلَامُ | ہمارا دین اسلام ہے |
| دُسْتُورُنَا الْقُرْآنُ | ہمارا دستور قرآن ہے |
| لُغَتُنَا الْعَرَبِيَّةُ | ہماری زبان عربی ہے |
| أَيْنَ أُمَّهَاتُكُمْ؟ | تمہاری امّائیں کہاں ہیں؟ |
| خَرَجْنَ مِنَ الْفَصْلِ | وہ کلاس سے نکلیں |
| ذَهَبْنَ إِلَى الْمُدِيرِ | وہ پرنسپل کی طرف گئیں |

## سوال و جواب — مکالمے کی مشق

| سوال | جواب |
|------|------|
| مَنْ أَنْتُمْ؟ | نَحْنُ طُلَّابٌ |
| مِنْ أَيْنَ أَنْتُمْ؟ | نَحْنُ مِنَ الْبَاكِسْتَانِ |
| مَا لُغَتُكُمْ؟ | لُغَتُنَا الأُرْدِيَّةُ |
| أَيْنَ مَدْرَسَتُكُمْ؟ | مَدْرَسَتُنَا هُنَا |
| أَيْنَ مُعَلِّمُكُمْ؟ | مُعَلِّمُنَا فِي الْفَصْلِ |
| هَلْ عِنْدَكُمْ سَيَّارَةٌ؟ | نَعَمْ، عِنْدَنَا سَيَّارَةٌ |

- تمام ضمائر جمع کے ساتھ ۳ اسماء کی مشق لکھیں
