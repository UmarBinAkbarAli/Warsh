## بسم اللہ الرحمٰن الرحیم

---

## پارٹ ۲ کا آغاز — بَيْتِي (میرا گھر)

پارٹ ۱ مکمل ہو گیا۔ پارٹ ۲ ایک مربوط پیراگراف کی شکل میں ہے — **میرا گھر**:

| عربی | ترجمہ |
|------|--------|
| بَيْتِي أَمَامَ الْمَسْجِدِ | میرا گھر مسجد کے سامنے ہے |
| بَيْتِي جَمِيلٌ وَصَغِيرٌ | میرا گھر خوبصورت اور چھوٹا ہے |
| فِيهِ غُرَفٌ | اس میں کمرے ہیں |
| هٰذِهِ غُرْفَتِي | یہ میرا کمرہ ہے |
| فِيهَا نَافِذَةٌ كَبِيرَةٌ | اس میں ایک بڑی کھڑکی ہے |
| هٰذَا سَرِيرِي وَهٰذِهِ كُرْسِيِّي | یہ میرا بستر ہے اور یہ میری کرسی ہے |
| وَهٰذَا مَكْتَبِي | اور یہ میری میز ہے |
| سَاعَتِي وَقَلَمِي وَكِتَابِي عَلَى الْمَكْتَبِ | میری گھڑی، قلم اور کتاب میز پر ہیں |
| وَحَقِيبَتِي تَحْتَ الْمَكْتَبِ | اور میرا بیگ میز کے نیچے ہے |
| نَافِذَةُ غُرْفَتِي مَفْتُوحَةٌ | میرے کمرے کی کھڑکی کھلی ہوئی ہے |
| تِلْكَ غُرْفَةُ أَخِي | وہ میرے بھائی کا کمرہ ہے |
| وَتِلْكَ غُرْفَةُ أُخْتِي | اور وہ میری بہن کا کمرہ ہے |
| غُرْفَةُ أَخِي كَبِيرَةٌ وَغُرْفَةُ أُخْتِي صَغِيرَةٌ | میرے بھائی کا کمرہ بڑا ہے اور میری بہن کا چھوٹا |
| غُرْفَةُ أَخِي أَمَامَ غُرْفَتِي | میرے بھائی کا کمرہ میرے کمرے کے سامنے ہے |
| وَغُرْفَةُ أُخْتِي أَمَامَ الْمَطْبَخِ | اور میری بہن کا کمرہ باورچی خانے کے سامنے ہے |
| أَخِي اسْمُهُ سَعْدٌ | میرے بھائی کا نام سعد ہے |
| أُحِبُّ وَالِدَيَّ وَأَخِي وَأُخْتِي | میں اپنے والدین، بھائی اور بہن سے محبت کرتا ہوں |

## سبق ۱۲ — مکالمہ: حال کیسے ہو؟

### تحیات اور حال پوچھنا

| عربی | ترجمہ |
|------|--------|
| كَيْفَ حَالُكَ؟ (مذکر) | تم کیسے ہو؟ |
| كَيْفَ حَالُكِ؟ (مؤنث) | تم کیسی ہو؟ |
| بِخَيْرٍ، وَالْحَمْدُ لِلَّهِ | اچھا ہوں، اللہ کا شکر ہے |
| مِنْ أَيْنَ أَنْتِ؟ | تم کہاں سے ہو؟ |
| أَنَا مِنَ الشَّامِ | میں شام (سیریا) سے ہوں |
| مَا اسْمُكِ؟ | تمہارا نام کیا ہے؟ |
| اسْمِي آمِنَةُ | میرا نام آمنہ ہے |

## مؤنث مخاطب ضمیر — كِ

مؤنث کو مخاطب کرتے وقت كَ کی جگہ **كِ** آتا ہے:

| مذکر | مؤنث | ترجمہ |
|------|------|--------|
| اسْمُكَ | اسْمُكِ | تمہارا نام |
| بَيْتُكَ | بَيْتُكِ | تمہارا گھر |
| كِتَابُكَ | كِتَابُكِ | تمہاری کتاب |
| أَعِنْدَكَ؟ | أَعِنْدَكِ؟ | کیا تمہارے پاس؟ |

---

## فعل کی مؤنث شکل — تاءِ تانیث

### قاعدہ

فعل کے آخر میں **تَ (ساکن)** لگانے سے مؤنث بن جاتا ہے:

| مذکر فاعل | فعل (مذکر) | مؤنث فاعل | فعل (مؤنث) |
|---------|-----------|---------|-----------|
| مُحَمَّدٌ | خَرَجَ | آمِنَةُ | خَرَجَتْ |
| اَلْمُعَلِّمُ | ذَهَبَ | اَلْمُعَلِّمَةُ | ذَهَبَتْ |
| اَلطَّالِبُ | جَلَسَ | اَلطَّالِبَةُ | جَلَسَتْ |
| اَلطَّبِيبُ | رَجَعَ | اَلطَّبِيبَةُ | رَجَعَتْ |
| أَبِي | خَرَجَ مِنَ الْبَيْتِ | أُمِّي | خَرَجَتْ مِنَ الْبَيْتِ |

### مثالیں

| مذکر | ترجمہ | مؤنث | ترجمہ |
|------|--------|------|--------|
| خَرَجَ مُحَمَّدٌ مِنَ الْبَيْتِ | محمد گھر سے نکلا | خَرَجَتْ آمِنَةُ مِنَ الْبَيْتِ | آمنہ گھر سے نکلی |
| ذَهَبَ الْمُعَلِّمُ إِلَى الْفَصْلِ | استاد کلاس کی طرف گیا | ذَهَبَتِ الْمُعَلِّمَةُ إِلَى الْفَصْلِ | استانی کلاس کی طرف گئی |
| جَلَسَ الطَّالِبُ فِي الْفَصْلِ | طالبِ علم کلاس میں بیٹھا | جَلَسَتِ الطَّالِبَةُ فِي الْفَصْلِ | طالبہ کلاس میں بیٹھی |
| خَرَجَ أَخِي مِنَ الْبَيْتِ | میرا بھائی گھر سے نکلا | خَرَجَتْ أُخْتِي مِنَ الْبَيْتِ | میری بہن گھر سے نکلی |

---

## دو ساکن جمع ہوں تو — کسرہ لگائیں

جب دو ساکن (سکون والے حروف) ملیں تو پہلے ساکن پر **کسرہ** لگائیں:

| مثال | وضاحت |
|-------|--------|
| خَرَجَتِ الطَّالِبَةُ | تَ ساکن + الف ساکن → تِ پڑھیں |
| ذَهَبَتِ الْمُعَلِّمَةُ | تَ ساکن + الف ساکن → تِ پڑھیں |
| مِنِ الْبَيْتِ | نون ساکن + الف ساکن → نِ پڑھیں |

---

## اسمِ موصول — اَلَّذِي اور اَلَّتِي

| | مذکر | مؤنث |
|--|------|------|
| اسمِ موصول | اَلَّذِي | اَلَّتِي |
| مطلب | جو (مرد/مذکر) | جو (عورت/مؤنث) |

### مثالیں

| عربی | ترجمہ |
|------|--------|
| الْكِتَابُ الَّذِي عَلَى الْمَكْتَبِ لِلْمُعَلِّمِ | وہ کتاب جو میز پر ہے استاد کی ہے |
| الْقَلَمُ الْمَكْسُورُ الَّذِي فِي حَقِيبَتِي | وہ ٹوٹا ہوا قلم جو میرے بیگ میں ہے |
| السَّيَّارَةُ الَّتِي خَرَجَتِ الْآنَ مِنَ الْمَدْرَسَةِ | وہ گاڑی جو ابھی اسکول سے نکلی |
| الْفَتَاةُ الَّتِي أَمَامَ الْمُعَلِّمِ | وہ لڑکی جو استاد کے سامنے ہے |
| الْبَطَّةُ الَّتِي تَحْتَ الشَّجَرَةِ | وہ بطخ جو درخت کے نیچے ہے |

## مُنَادَا — پکارنے کا طریقہ (یاد دہانی)

جب **يَا** سے پکاریں تو جس کو پکاریں (مُنَادَا) سے تنوین ختم ہو جاتی ہے:

| بغیر يَا | يَا کے ساتھ | ترجمہ |
|---------|------------|-------|
| أُمِّي | يَا أُمِّي | اے ماں! |
| أَبِي | يَا أَبِي | اے ابا! |
| مُحَمَّدٌ | يَا مُحَمَّدُ | اے محمد! |
| آمِنَةُ | يَا آمِنَةُ | اے آمنہ! |

**مذکر اور مؤنث مخاطب میں فرق:**

| | مذکر | مؤنث |
|--|------|------|
| أَيْنَ قَلَمُ__؟ | أَيْنَ قَلَمُكَ يَا مُحَمَّدُ؟ | أَيْنَ قَلَمُكِ يَا آمِنَةُ؟ |
| أَبُو__ | أَبُوكَ | أَبُوكِ |

## خاندان کی اصطلاحات

### باپ کی طرف سے

| عربی | اردو |
|------|------|
| عَمٌّ | چچا (باپ کا بھائی) |
| عَمَّةٌ | پھوپھی (باپ کی بہن) |

### ماں کی طرف سے

| عربی | اردو |
|------|------|
| خَالٌ | ماموں (ماں کا بھائی) |
| خَالَةٌ | خالہ (ماں کی بہن) |

### دیگر رشتے

| عربی | اردو |
|------|------|
| اِبْنٌ / وَلَدٌ | بیٹا |
| بِنْتٌ | بیٹی |
| أَخٌ | بھائی |
| أُخْتٌ | بہن |
| زَوْجٌ | شوہر |
| زَوْجَةٌ | بیوی |
| اِبْنُ الْعَمِّ | چچا کا بیٹا (کزن) |
| بِنْتُ الْخَالِ | ماموں کی بیٹی |

## مفعول بہ — نصب کی یاد دہانی

جو چیز فعل کا مفعول ہو وہ **نصب** میں آتی ہے:

| فعل | فاعل | مفعول (نصب میں) |
|-----|------|----------------|
| أُحِبُّ | أَنَا | وَالِدَيَّ |
| أُحِبُّ | أَنَا | اللهَ |
| أُحِبُّ | أَنَا | النَّبِيَّ ﷺ |
| أُحِبُّ | أَنَا | اللُّغَةَ الْعَرَبِيَّةَ |

> **نوٹ:** أُحِبُّ = میں محبت کرتا ہوں — اس کے بعد مفعول نصب میں آتا ہے

## مشق (تَمَارِيْن)

### مشق الف — فعل کو مؤنث بنائیں

| مذکر فعل | مؤنث فعل |
|---------|---------|
| خَرَجَ مُحَمَّدٌ | خَرَجَتْ آمِنَةُ |
| ذَهَبَ أَخِي | _______ أُخْتِي |
| جَلَسَ الطَّالِبُ | _______ الطَّالِبَةُ |
| رَجَعَ الْوَالِدُ | _______ الْوَالِدَةُ |

### مشق ب — اَلَّذِي یا اَلَّتِي؟

| جملہ | درست انتخاب |
|------|------------|
| اَلْكِتَابُ _____ عَلَى الْمَكْتَبِ | اَلَّذِي |
| السَّيَّارَةُ _____ خَرَجَتْ | اَلَّتِي |
| الْوَلَدُ _____ فِي الْمَسْجِدِ | اَلَّذِي |
| الْبِنْتُ _____ أَمَامَ الْمُعَلِّمِ | اَلَّتِي |

### مشق ج — مُنَادَا بدلیں

مذکر جملہ مؤنث میں بدلیں:

- أَيْنَ كِتَابُكَ يَا مُحَمَّدُ؟
- بدلیں: أَيْنَ كِتَابُكِ يَا آمِنَةُ؟

## ہوم ورک

- اپنے گھر کے بارے میں ۱۰ جملے لکھیں (بَيْتِي کے موضوع پر)
- فعل کی مؤنث شکل کے ۵ جملے بنائیں
- خاندان کے رشتوں کی مکمل فہرست یاد کریں
- مشقیں **لکھ کر** مکمل کریں
