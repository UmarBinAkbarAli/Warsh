# ریڈر لیکچر ۱۱ — سبق ۱۶ اور ۱۷: گنتی اور تثنیہ

## بسم اللہ الرحمٰن الرحیم

---

## گنتی — ۳ سے ۱۰ تک

| عدد | مذکر شکل | مؤنث شکل |
|-----|---------|---------|
| ۳ | ثَلَاثَةٌ | ثَلَاثٌ |
| ۴ | أَرْبَعَةٌ | أَرْبَعٌ |
| ۵ | خَمْسَةٌ | خَمْسٌ |
| ۶ | سِتَّةٌ | سِتٌّ |
| ۷ | سَبْعَةٌ | سَبْعٌ |
| ۸ | ثَمَانِيَةٌ | ثَمَانٍ |
| ۹ | تِسْعَةٌ | تِسْعٌ |
| ۱۰ | عَشَرَةٌ | عَشْرٌ |

---

## عدد اور معدود — بنیادی قواعد

### ۱. مُرَكَّبِ اِضَافِي کی شکل

| حصہ | اصطلاح | حالت |
|-----|---------|------|
| گنتی (پہلا) | عَدَد | مُضَاف |
| جو گنا جائے (دوسرا) | مَعْدُود | مُضَافٌ إِلَيْه — جر میں |

### ۲. جنس مخالف کا قاعدہ

| مَعْدُود | عَدَد |
|---------|------|
| مذکر | مؤنث شکل آئے گی |
| مؤنث | مذکر شکل آئے گی |

---

## مثالیں — جنس مخالف

| عربی | ترجمہ | نوٹ |
|------|--------|-----|
| خَمْسَةُ طُلَّابٍ | پانچ طالبِ علم | طُلَّاب مذکر → خَمْسَةٌ مؤنث |
| خَمْسُ طَالِبَاتٍ | پانچ طالبات | طالبات مؤنث → خَمْسٌ مذکر |
| ثَلَاثَةُ رِجَالٍ | تین آدمی | رجال مذکر → ثَلَاثَةٌ مؤنث |
| ثَلَاثُ بَنَاتٍ | تین بیٹیاں | بنات مؤنث → ثَلَاثٌ مذکر |
| أَرْبَعَةُ أَبْوَابٍ | چار دروازے | مذکر → أَرْبَعَةٌ |
| أَرْبَعُ أَخَوَاتٍ | چار بہنیں | مؤنث → أَرْبَعٌ |
| سَبْعَةُ أَبْنَاءٍ | سات بیٹے | مذکر → سَبْعَةٌ |
| سَبْعُ بَنَاتٍ | سات بیٹیاں | مؤنث → سَبْعٌ |
| عَشَرَةُ طُلَّابٍ | دس طالبِ علم | مذکر → عَشَرَةٌ |
| عَشْرُ طَالِبَاتٍ | دس طالبات | مؤنث → عَشْرٌ |
| ثَمَانِيَةُ مُدَرِّسِينَ | آٹھ استاد | مذکر → ثَمَانِيَةٌ |
| ثَمَانِي مُدَرِّسَاتٍ | آٹھ استانیاں | مؤنث → ثَمَانٍ |

---

## سبق ۱۶ کے جملے

| عربی | ترجمہ |
|------|--------|
| فِي بَيْتِنَا ثَلَاثُ غُرَفٍ | ہمارے گھر میں تین کمرے ہیں |
| فِي الْجَامِعَةِ عَشَرَةُ حَافِلَاتٍ | یونیورسٹی میں دس بسیں ہیں |
| فِي هٰذِهِ الْمَدْرَسَةِ ثَمَانِي مُدَرِّسَاتٍ | اس مدرسے میں آٹھ استانیاں ہیں |
| لِعَبَّاسٍ سَبْعُ بَنَاتٍ | عباس کی سات بیٹیاں ہیں |
| فِي بَيْتِنَا تِسْعُ دَجَاجَاتٍ | ہمارے گھر میں نو مرغیاں ہیں |
| فِي الْجَامِعَةِ خَمْسُ كُلِّيَّاتٍ | یونیورسٹی میں پانچ کالج ہیں |
| فِي مُسْتَشْفَى الْوِلَادَةِ عَشْرُ طَبِيبَاتٍ | زچہ خانے میں دس لیڈی ڈاکٹر ہیں |
| لِخَالِدٍ ثَلَاثَةُ أَبْنَاءٍ وَأَرْبَعُ بَنَاتٍ | خالد کے تین بیٹے اور چار بیٹیاں ہیں |

---

## كَمْ — کتنے؟

**كَمْ** کے بعد آنے والا اسم **تَمْيِيز** کہلاتا ہے — واحد، نکرہ، نصب:

| سوال | جواب |
|------|------|
| كَمْ أَخًا لَكَ؟ | لِي خَمْسَةُ إِخْوَةٍ |
| كَمْ أُخْتًا لَكَ؟ | لِي ثَلَاثُ أَخَوَاتٍ |
| كَمْ بَابًا لِلسَّيَّارَةِ؟ | لَهَا أَرْبَعَةُ أَبْوَابٍ |
| كَمْ طَالِبًا فِي فَصْلِكَ؟ | فِي فَصْلِي عَشَرَةُ طُلَّابٍ |
| كَمْ شَهْرًا فِي السَّنَةِ؟ | فِيهَا اثْنَا عَشَرَ شَهْرًا |
| كَمْ حَقْلًا عِنْدَكَ؟ | عِنْدِي عَشَرَةُ حُقُولٍ |

---

## تثنیہ — دو کے لیے

### تثنیہ کیسے بنائیں

| حالت | لاحقہ | مثال | ترجمہ |
|------|-------|-------|--------|
| رَفْع | ـَانِ | كِتَابَانِ | دو کتابیں |
| نَصْب / جَر | ـَيْنِ | كِتَابَيْنِ | دو کتابوں کو/سے |

### مثالیں

| واحد | تثنیہ (رفع) | تثنیہ (نصب/جر) | ترجمہ |
|------|------------|--------------|-------|
| قَلَمٌ | قَلَمَانِ | قَلَمَيْنِ | دو قلم |
| طَالِبٌ | طَالِبَانِ | طَالِبَيْنِ | دو طالبِ علم |
| طَالِبَةٌ | طَالِبَتَانِ | طَالِبَتَيْنِ | دو طالبات |
| سَيَّارَةٌ | سَيَّارَتَانِ | سَيَّارَتَيْنِ | دو گاڑیاں |
| مُعَلِّمٌ | مُعَلِّمَانِ | مُعَلِّمَيْنِ | دو استاد |
| كِتَابٌ | كِتَابَانِ | كِتَابَيْنِ | دو کتابیں |
| يَدٌ | يَدَانِ | يَدَيْنِ | دو ہاتھ |
| عَيْنٌ | عَيْنَانِ | عَيْنَيْنِ | دو آنکھیں |
| أُذُنٌ | أُذُنَانِ | أُذُنَيْنِ | دو کان |
| مِفْتَاحٌ | مِفْتَاحَانِ | مِفْتَاحَيْنِ | دو چابیاں |
| مُدَرِّسٌ | مُدَرِّسَانِ | مُدَرِّسَيْنِ | دو استاد |
| نَافِذَةٌ | نَافِذَتَانِ | نَافِذَتَيْنِ | دو کھڑکیاں |

## سبق ۱۷ — تثنیہ کے جملے

| عربی | ترجمہ |
|------|--------|
| فِي غُرْفَتِي نَافِذَتَانِ كَبِيرَتَانِ | میرے کمرے میں دو بڑی کھڑکیاں ہیں |
| هٰذَانِ كِتَابَانِ لِمَنْ هُمَا؟ | یہ دونوں کتابیں کس کی ہیں؟ |
| هُمَا لِي | یہ دونوں میری ہیں |
| هٰذَانِ طَبِيبَانِ مِنْ إِنْكِلْتَرَا | یہ دونوں ڈاکٹر انگلینڈ سے ہیں |
| عِنْدَ إِبْرَاهِيمَ سَيَّارَتَانِ كَبِيرَتَانِ وَسَيَّارَتَانِ صَغِيرَتَانِ | ابراہیم کے پاس دو بڑی اور دو چھوٹی گاڑیاں ہیں |
| لَنَا أُذُنَانِ وَعَيْنَانِ | ہمارے دو کان اور دو آنکھیں ہیں |
| فِي هٰذَا الْمَحَلَّةِ مَسْجِدَانِ | اس محلے میں دو مسجدیں ہیں |

---

## مشق کے جملے — سوال و جواب

| سوال | جواب |
|------|------|
| كَمْ أَخًا لَكَ يَا مُحَمَّدُ؟ | لِي خَمْسَةُ إِخْوَةٍ |
| وَكَمْ أُخْتًا؟ | لِي ثَلَاثُ أَخَوَاتٍ |
| كَمْ بِنْتًا لَكَ يَا أُسَامَةُ؟ | لِي سَبْعُ بَنَاتٍ |
| كَمْ ابْنًا لَكَ؟ | لِي أَرْبَعَةُ أَبْنَاءٍ |
| كَمْ عَمًّا لَكَ؟ | لِي أَرْبَعَةُ أَعْمَامٍ |
| كَمْ سَيَّارَةً عِنْدَكَ؟ | عِنْدِي سَيَّارَتَانِ كَبِيرَةٌ وَصَغِيرَةٌ |

## ہوم ورک

- گنتی ۳ تا ۱۰ کے ساتھ اپنے خاندان کے بارے میں جملے بنائیں
- تثنیہ کی ۱۰ مثالیں لکھیں (رفع اور نصب دونوں)
- كَمْ کے ۵ سوال اور جواب لکھیں
