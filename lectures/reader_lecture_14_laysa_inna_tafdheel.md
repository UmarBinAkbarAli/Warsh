## لَيْسَ — فعل ناقص (نہیں ہے)

### تعارف

**لَيْسَ** کا مطلب ہے "نہیں ہے" — یہ **فعل ناقص** ہے:
- **فعل ناقص** = اس میں زمانہ ہے لیکن مصدری معنی نہیں
- جیسے كَانَ (تھا)، لَيْسَ (نہیں ہے)

**لَيْسَ کا قاعدہ:**
1. اپنے بعد آنے والے اسم کو **نصب** دیتا ہے (اِسمُ لَيْسَ)
2. خبر کے ساتھ اکثر **بَ** (حرفِ جار) آتا ہے — پھر خبر جر میں

### لَيْسَ کی مکمل گردان

| صیغہ | لَيْسَ | مثال | ترجمہ |
|------|-------|-------|--------|
| هُوَ (مذکر غائب) | لَيْسَ | لَيْسَ مُسْلِمًا | وہ مسلمان نہیں |
| هِيَ (مؤنث غائب) | لَيْسَتْ | لَيْسَتْ مُسْلِمَةً | وہ مسلمان نہیں |
| هُمَا (تثنیہ) | لَيْسَا | لَيْسَا هُنَا | وہ دونوں یہاں نہیں |
| هُمْ (جمع مذکر) | لَيْسُوا | لَيْسُوا طُلَّابًا | وہ طالبِ علم نہیں |
| هُنَّ (جمع مؤنث) | لَسْنَ | لَسْنَ مُتَزَوِّجَاتٍ | وہ شادی شدہ نہیں |
| أَنْتَ | لَسْتَ | لَسْتَ كَبِيرًا | تم بڑے نہیں |
| أَنْتِ | لَسْتِ | لَسْتِ فَقِيرَةً | تم غریب نہیں |
| أَنْتُمَا | لَسْتُمَا | — | — |
| أَنْتُمْ | لَسْتُمْ | لَسْتُمْ بِجُدُدٍ | تم سب نئے نہیں |
| أَنْتُنَّ | لَسْتُنَّ | لَسْتُنَّ بِمُجْتَهِدَاتٍ | تم سب محنتی نہیں |
| أَنَا | لَسْتُ | لَسْتُ بِمُدَرِّسٍ | میں مدرس نہیں |
| نَحْنُ | لَسْنَا | لَسْنَا بِطُلَّابٍ | ہم طالبِ علم نہیں |

### لَيْسَ — مثبت اور منفی

| مثبت | منفی (لَيْسَ) |
|------|------------|
| هُوَ تَاجِرٌ | لَيْسَ بِتَاجِرٍ |
| هِيَ طَبِيبَةٌ | لَيْسَتْ بِطَبِيبَةٍ |
| الطُّلَّابُ صِغَارٌ | الطُّلَّابُ لَيْسُوا بِصِغَارٍ |
| آمِنَةُ طَبِيبَةٌ | آمِنَةُ لَيْسَتْ بِطَبِيبَةٍ |
| الْفَتَيَاتُ مُتَزَوِّجَاتٌ | الْفَتَيَاتُ لَسْنَ بِمُتَزَوِّجَاتٍ |
| أَنْتَ كَبِيرٌ | لَسْتَ بِكَبِيرٍ |
| أَنْتُمْ جُدُدٌ | لَسْتُمْ بِجُدُدٍ |
| أَنَا مُدَرِّسٌ | لَسْتُ بِمُدَرِّسٍ |
| نَحْنُ طُلَّابٌ | لَسْنَا بِطُلَّابٍ |
| الْبَابُ مَفْتُوحٌ | لَيْسَ الْبَابُ بِمَفْتُوحٍ |
| الْمَاءُ بَارِدٌ | لَيْسَ الْمَاءُ بِبَارِدٍ |

> **یاد رہے:** غیر منصرف اسم اگر بَ کے بعد آئے تو جر میں فتحہ رہے گا
> مثال: لَيْسَ الطِّفْلُ بِجَوْعَانَ (جوعان غیر منصرف — فتحہ)

### فعل ناقص — اسم ظاہر ہو تو واحد آتا ہے

جب لَيْسَ کے بعد اسمِ ظاہر آئے تو لَيْسَ کا صیغہ **واحد** ہوتا ہے:

| | غلط | درست |
|--|-----|------|
| مؤنث اسم ظاہر | لَسْنَ أَخَوَاتِي طَالِبَاتٌ | لَيْسَتْ أَخَوَاتِي طَالِبَاتٌ |
| جمع اسم ظاہر | لَيْسُوا الطُّلَّابُ هُنَا | لَيْسَ الطُّلَّابُ هُنَا |

> **قاعدہ (فعل کا عام قاعدہ):**
> - فعل پہلے آئے → ہمیشہ واحد (صرف جنس کا فرق)
> - اسم پہلے آئے → اسم کے مطابق

## إِنَّ کے ساتھ — مبتدا مؤخر اور خبر مقدم

### خبر مقدم اور مبتدا مؤخر

عربی میں اگر خبر **جار مجرور** ہو تو اسے پہلے لایا جاتا ہے:

| ترتیب | مثال | ترجمہ |
|-------|-------|--------|
| عام ترتیب | زَيْدٌ فِي الدَّارِ | زید گھر میں ہے |
| خبر مقدم | فِي الدَّارِ زَيْدٌ | گھر میں زید ہے |

اصطلاحات:
- **خبر مقدم** = وہ خبر جسے آگے لایا گیا
- **مبتدا مؤخر** = وہ مبتدا جو پیچھے چلا گیا

### إِنَّ کے ساتھ — اسم إِنَّ کہاں بھی ہو نصب ملتی ہے

| عربی | ترجمہ |
|------|--------|
| إِنَّ فِي الْفَصْلِ خَمْسَةَ طُلَّابٍ جُدُدٍ | بے شک کلاس میں پانچ نئے طالبِ علم ہیں |
| إِنَّ لِي ثَلَاثَ أَخَوَاتٍ | بے شک میری تین بہنیں ہیں |
| إِنَّ لَنَا مُدَرِّسًا جَيِّدًا | بے شک ہمارے ایک اچھے استاد ہیں |
| إِنَّ فِي جَيْبِي مِائَةَ رِيَالٍ | بے شک میری جیب میں سو ریال ہیں |
| إِنَّ فِي الْهِنْدِ أَنْهَارًا كَثِيرَةً | بے شک ہندوستان میں بہت نہریں ہیں |
| إِنَّ لَكَ بَرْقِيَّةً فِي مَكْتَبِ الْبَرِيدِ | بے شک پوسٹ آفس میں آپ کے لیے ایک ٹیلیگرام ہے |
| إِنَّ لِي أَلْفَ دُولَارٍ فِي الْمَصْرِفِ | بے شک میرے بینک میں ایک ہزار ڈالر ہیں |

## سبق ۲ — تفضیل (کمپیریٹو اور سپرلیٹو)

### أَفْعَل کا وزن — دو کام

**أَفْعَل** کے وزن پر آنے والے اسم دو مقاصد کے لیے آتے ہیں:

| مقصد | استعمال | مثال |
|------|---------|-------|
| کمپیریٹو (زیادہ) | أَفْعَل + مِنْ | حَامِدٌ أَطْوَلُ مِنْ زَيْدٍ — حامد زید سے لمبا ہے |
| سپرلیٹو (سب سے) | أَفْعَل مُضَاف | أَطْوَلُ الطُّلَّابِ — طالبِ علموں میں سب سے لمبا |

---

### کمپیریٹو — مِنْ کے ساتھ

| عربی | ترجمہ |
|------|--------|
| حَامِدٌ أَطْوَلُ مِنْ زَيْدٍ | حامد زید سے لمبا ہے |
| خَطِّي أَجْمَلُ مِنْ خَطِّكَ | میری لکھائی تمہاری لکھائی سے خوبصورت ہے |
| هٰذَا الْمَسْكَنُ أَبْعَدُ مِنْ مَسْكَنِكُمْ | یہ ہاسٹل تم سب کے ہاسٹل سے دور ہے |
| هُوَ أَصْغَرُ مِنِّي | وہ مجھ سے چھوٹا ہے |
| الطُّلَّابُ فِي فَصْلِنَا أَكْثَرُ | ہماری کلاس میں طالبِ علم زیادہ ہیں |

### سپرلیٹو — مُضَاف کے طور پر

| عربی | ترجمہ |
|------|--------|
| أَطْوَلُ الطُّلَّابِ إِبْرَاهِيمُ | طالبِ علموں میں سب سے لمبا ابراہیم ہے |
| أَكْبَرُ طَالِبٍ فِي فَصْلِنَا | ہماری کلاس کا سب سے بڑا طالبِ علم |
| الْمَسْكَنُ الْخَامِسُ أَحْسَنُ | پانچواں ہاسٹل سب سے بہتر ہے |
| غُرَفُهُ أَوْسَعُ | اس کے کمرے زیادہ کشادہ ہیں |
| نَوَافِذُهُ أَكْبَرُ | اس کی کھڑکیاں زیادہ بڑی ہیں |
| مَرَاحِيضُهُ أَنْظَفُ | اس کے واش روم زیادہ صاف ہیں |
| سُرُرُهُ أَجْمَلُ | اس کے بستر زیادہ اچھے ہیں |

### مکالمہ — سبق ۳ (احمد اور علی)

| عربی | ترجمہ |
|------|--------|
| كَمْ طَالِبًا فِي فَصْلِكُمْ يَا عَلِيُّ؟ | علی، تمہاری کلاس میں کتنے طالبِ علم ہیں؟ |
| فِي فَصْلِنَا أَرْبَعَةَ عَشَرَ طَالِبًا | ہماری کلاس میں ۱۴ طالبِ علم ہیں |
| الطُّلَّابُ فِي فَصْلِنَا أَكْثَرُ | ہماری کلاس میں طالبِ علم زیادہ ہیں |
| فِيهِ تِسْعَةَ عَشَرَ طَالِبًا | اس میں ۱۹ طالبِ علم ہیں |
| مَا اسْمُ الطَّالِبِ الْجَدِيدِ الَّذِي جَاءَ أَمْسِ؟ | کل آنے والے نئے طالبِ علم کا نام کیا ہے؟ |
| اسْمُهُ أُسَامَةُ | اس کا نام اسامہ ہے |
| هُوَ طَوِيلٌ جِدًّا | وہ بہت لمبا ہے |
| بَلَى، لَكِنَّ حَامِدًا أَطْوَلُ مِنْهُ | ہاں، لیکن حامد اس سے بھی لمبا ہے |
| بَلِ هُوَ أَطْوَلُ الطُّلَّابِ | بلکہ وہ سب سے لمبا طالبِ علم ہے |
| أَهٰذَا دَفْتَرُكَ يَا أَحْمَدُ؟ | احمد، کیا یہ آپ کی کاپی ہے؟ |
| نَعَمْ، إِنَّ خَطَّكَ جَمِيلٌ جِدًّا | ہاں — بے شک آپ کی لکھائی بہت خوبصورت ہے |
| شُكْرًا يَا عَلِيُّ، خَطِّي جَمِيلٌ وَخَطُّكَ أَجْمَلُ مِنْهُ | شکریہ، میری لکھائی خوبصورت ہے اور آپ کی اس سے زیادہ |
| أَنَا فِي الْمَسْكَنِ الْخَامِسِ | میں پانچویں ہاسٹل میں ہوں |
| وَهُوَ بَعِيدٌ جِدًّا عَنِ الْمَدْرَسَةِ | اور وہ مدرسے سے بہت دور ہے |
| أَنَا فِي الْمَسْكَنِ الثَّامِنِ | میں آٹھویں ہاسٹل میں ہوں |
| وَهُوَ أَبْعَدُ مِنْ مَسْكَنِكُمْ | اور وہ تمہارے ہاسٹل سے زیادہ دور ہے |
| أَيُّهُمَا أَحْسَنُ؟ | ان دونوں میں سے کون سا بہتر ہے؟ |
| الْمَسْكَنُ الْخَامِسُ أَحْسَنُ | پانچواں ہاسٹل بہتر ہے |

| ۱۱ | أَحَدَ عَشَرَ | إِحْدَى عَشْرَةَ |
| ۱۲ | اثْنَا عَشَرَ | اثْنَتَا عَشْرَةَ |
| ۱۳ | ثَلَاثَةَ عَشَرَ | ثَلَاثَ عَشْرَةَ |
| ۱۴ | أَرْبَعَةَ عَشَرَ | أَرْبَعَ عَشْرَةَ |
| ۱۵ | خَمْسَةَ عَشَرَ | خَمْسَ عَشْرَةَ |
| ۱۶ | سِتَّةَ عَشَرَ | سِتَّ عَشْرَةَ |
| ۱۷ | سَبْعَةَ عَشَرَ | سَبْعَ عَشْرَةَ |
| ۱۸ | ثَمَانِيَةَ عَشَرَ | ثَمَانِيَ عَشْرَةَ |
| ۱۹ | تِسْعَةَ عَشَرَ | تِسْعَ عَشْرَةَ |

---

### ۱۱ تا ۱۹ کے قواعد

| قاعدہ | تفصیل |
|-------|--------|
| معدود کی حالت | نصب (۱۱ سے ۹۹ تک) |
| معدود کا عدد | واحد (نکرہ) |
| ۱۱ اور ۱۲ میں | عدد اور معدود جنس میں ایک جیسے |
| ۱۳ سے ۱۹ تک | عدد کا **پہلا حصہ** اور معدود — جنس مخالف |
| ۱۳-۱۹ مبنی | تینوں حالتوں میں فتحہ پر ثابت |

### مثالیں

| عربی | ترجمہ | نوٹ |
|------|--------|-----|
| أَحَدَ عَشَرَ طَالِبًا | گیارہ طالبِ علم | دونوں مذکر — ایک جیسے |
| اثْنَا عَشَرَ طَالِبًا | بارہ طالبِ علم | — |
| ثَلَاثَةَ عَشَرَ طَالِبًا | تیرہ طالبِ علم | طالب مذکر → ثَلَاثَةَ مؤنث |
| أَرْبَعَةَ عَشَرَ طَالِبًا | چودہ طالبِ علم | طالب مذکر → أَرْبَعَةَ مؤنث |
| خَمْسَةَ عَشَرَ طَالِبًا | پندرہ طالبِ علم | مذکر → مؤنث عدد |
| تِسْعَةَ عَشَرَ طَالِبًا | انیس طالبِ علم | — |
| إِحْدَى عَشْرَةَ طَالِبَةً | گیارہ طالبات | دونوں مؤنث |
| ثَلَاثَ عَشْرَةَ طَالِبَةً | تیرہ طالبات | طالبہ مؤنث → ثَلَاثَ مذکر |

> **ترکیب:** عدد کا پہلا حصہ + عَشَرَ/عَشْرَةَ + معدود (نصب، واحد)

## فعل — اہم قاعدہ (جملہ فعلیہ میں)

اگر فعل پہلے آئے تو فعل **ہمیشہ واحد** ہوتا ہے:

| | اسم پہلے (جملہ اسمیہ) | فعل پہلے (جملہ فعلیہ) |
|--|---------------------|---------------------|
| واحد | اَلْمُسْلِمُ جَاءَ | جَاءَ الْمُسْلِمُ |
| تثنیہ | اَلْمُسْلِمَانِ جَاءَا | جَاءَ الْمُسْلِمَانِ |
| جمع | اَلْمُسْلِمُونَ جَاءُوا | جَاءَ الْمُسْلِمُونَ |
| مؤنث | اَلْمُسْلِمَةُ جَاءَتْ | جَاءَتِ الْمُسْلِمَةُ |
| تثنیہ مؤنث | اَلْمُسْلِمَتَانِ جَاءَتَا | جَاءَتِ الْمُسْلِمَتَانِ |
| جمع مؤنث | اَلْمُسْلِمَاتُ جِئْنَ | جَاءَتِ الْمُسْلِمَاتُ |

|--|------------|-----------|
| آغاز | اسم سے | فعل سے |
| مثال | زَيْدٌ قَائِمٌ | قَامَ زَيْدٌ |

| استعمال | مستقل صفت، حال | کوئی واقعہ ہونا |

## ہوم ورک

- لَيْسَ کی مکمل گردان یاد کریں
- لَيْسَ والے ۱۰ جملے بنائیں
- گنتی ۱۱ تا ۱۹ مذکر اور مؤنث دونوں کے ساتھ لکھیں
- کمپیریٹو اور سپرلیٹو کے ۵ جملے بنائیں
- جملہ اسمیہ اور جملہ فعلیہ کا فرق ایک مثال سے سمجھائیں

## خلاصہ

| موضوع | اہم نکتہ |
|-------|----------|
| لَيْسَ | فعل ناقص — نہیں ہے — خبر بَ کے ساتھ جر میں |
| لَيْسَ + اسم ظاہر | واحد صیغہ (مذکر/مؤنث کے مطابق) |
| مبتدا مؤخر | خبر جار مجرور ہو تو پہلے آتی ہے |
| إِنَّ + مبتدا مؤخر | أسم إِنَّ جہاں بھی ہو — نصب ملتی ہے |
| أَفْعَل (کمپیریٹو) | أَفْعَل + مِنْ |
| أَفْعَل (سپرلیٹو) | أَفْعَل مُضَاف |
| گنتی ۱۱-۱۹ | معدود واحد نصب — ۱۱-۱۲ سیم جنس — ۱۳-۱۹ پہلا حصہ مخالف جنس |
| فعل پہلے آئے | ہمیشہ واحد — صرف جنس کا فرق |
| جملہ اسمیہ بمقابلہ فعلیہ | استمرار بمقابلہ حدوث |
