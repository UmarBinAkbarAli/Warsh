# ریڈر لیکچر ۷ — سبق ۱۰: متصل ضمائر اور مُنْصَرِف / غیر مُنْصَرِف

## بسم اللہ الرحمٰن الرحیم

---

## گزشتہ سبق سے مکالمہ — سبق ۱۰ کا آغاز

یہ سبق مکالمے (Dialogue) کی شکل میں ہے — محمد اور ہمزہ کی گفتگو:

| عربی | ترجمہ |
|------|--------|
| مَنْ أَنْتَ؟ | آپ کون ہیں؟ |
| أَنَا مُحَمَّدٌ، طَالِبٌ فِي الْجَامِعَةِ | میں محمد ہوں — یونیورسٹی میں طالبِ علم |
| أَنَا طَالِبٌ جَدِيدٌ | میں نیا طالبِ علم ہوں |
| أَنَا مِنَ الْهِنْدِ | میں ہندوستان سے ہوں |
| مَا اسْمُكَ؟ | تمہارا نام کیا ہے؟ |
| اسْمِي مُحَمَّدٌ | میرا نام محمد ہے |
| مَا لُغَتُكَ؟ | تمہاری زبان کیا ہے؟ |
| لُغَتِي الْأُرْدِيَّةُ | میری زبان اردو ہے |
| أَهِيَ لُغَةٌ سَهْلَةٌ؟ | کیا وہ آسان زبان ہے؟ |
| نَعَمْ، هِيَ لُغَةٌ سَهْلَةٌ | ہاں، وہ آسان زبان ہے |
| أَيْنَ وَالِدُكَ؟ | تمہارے والد کہاں ہیں؟ |
| وَالِدِي فِي الْكُوَيْتِ، طَبِيبٌ شَهِيرٌ | میرے والد کویت میں ہیں — مشہور ڈاکٹر |
| أَعِنْدَكَ سَيَّارَةٌ؟ | کیا تمہارے پاس گاڑی ہے؟ |
| لَا، مَا عِنْدِي سَيَّارَةٌ | نہیں، میرے پاس گاڑی نہیں |
| عِنْدَهُ سَيَّارَةٌ | اس کے پاس گاڑی ہے |

## متصل ضمائر — اسم کے ساتھ

ضمیر جب اسم کے آخر میں لگے تو **متصل ضمیر** کہلاتی ہے۔ یہ **ملکیت** ظاہر کرتی ہے:

### یاء (ی) کے ساتھ — میرا / میری

| اسم | متصل ضمیر | ترجمہ |
|-----|---------|--------|
| كِتَابٌ | كِتَابِي | میری کتاب |
| قَلَمٌ | قَلَمِي | میرا قلم |
| بَيْتٌ | بَيْتِي | میرا گھر |
| أُمٌّ | أُمِّي | میری ماں |
| أَبٌ | أَبِي | میرے ابا |
| اسْمٌ | اسْمِي | میرا نام |
| لُغَةٌ | لُغَتِي | میری زبان |
| وَالِدٌ | وَالِدِي | میرے والد |

### کاف (ک) کے ساتھ — تیرا / تیری

| | مذکر مخاطب | مؤنث مخاطب |
|--|------------|------------|
| علامت | كَ | كِ |
| مثال | كِتَابُكَ | كِتَابُكِ |
| ترجمہ | تیری کتاب | تیری کتاب |

### ہاء (ہ / ہا) کے ساتھ — اس کا / اس کی

| | مذکر غائب | مؤنث غائب |
|--|-----------|-----------|
| علامت | هُ / هِ | هَا |
| مثال | كِتَابُهُ | كِتَابُهَا |
| ترجمہ | اس (مرد) کی کتاب | اس (عورت) کی کتاب |

---

## مکمل جدول — ایک اسم کے ساتھ تمام ضمائر

مثال: **كِتَابٌ**

| ضمیر | متصل | ترجمہ |
|------|------|--------|
| أَنَا (میں) | كِتَابِي | میری کتاب |
| أَنْتَ (تم — مذکر) | كِتَابُكَ | تیری کتاب |
| أَنْتِ (تم — مؤنث) | كِتَابُكِ | تیری کتاب |
| هُوَ (وہ — مذکر) | كِتَابُهُ | اس کی کتاب |
| هِيَ (وہ — مؤنث) | كِتَابُهَا | اس کی کتاب |

**دیگر اسماء کی مشق:**

| اسم | ی (میرا) | كَ (تیرا) | هُ (اس کا) | هَا (اس کی) |
|-----|---------|---------|---------|---------|
| سَرِيرٌ | سَرِيرِي | سَرِيرُكَ | سَرِيرُهُ | سَرِيرُهَا |
| مِفْتَاحٌ | مِفْتَاحِي | مِفْتَاحُكَ | مِفْتَاحُهُ | مِفْتَاحُهَا |
| يَدٌ | يَدِي | يَدُكَ | يَدُهُ | يَدُهَا |
| اِبْنٌ | اِبْنِي | اِبْنُكَ | اِبْنُهُ | اِبْنُهَا |

---

## لامِ متصل (لِي / لَكَ / لَهُ)

لام بھی ضمیر کے ساتھ آتا ہے — ملکیت یا تعلق کے لیے:

| | متصل | ترجمہ |
|--|------|--------|
| میرے لیے / میرا | لِي | لِي كِتَابٌ = میرے پاس کتاب ہے |
| تیرے لیے / تیرا | لَكَ | لَكَ كِتَابٌ = تیرے پاس کتاب ہے |
| اس کے لیے / اس کا | لَهُ | لَهُ كِتَابٌ = اس کے پاس کتاب ہے |
| اس کے لیے (مؤنث) | لَهَا | لَهَا كِتَابٌ = اس کے پاس کتاب ہے |

---

## عِنْدَ اور مَعَ — دو اہم ظرف

### عِنْدَ (پاس ہونا — Possession)

**عِنْدَ** = کے پاس (چیز ہو تو)

| عربی | ترجمہ |
|------|--------|
| عِنْدِي كِتَابٌ | میرے پاس کتاب ہے |
| عِنْدَكَ قَلَمٌ | تیرے پاس قلم ہے |
| عِنْدَهُ سَيَّارَةٌ | اس کے پاس گاڑی ہے |
| مَا عِنْدِي دَرَّاجَةٌ | میرے پاس سائیکل نہیں |

### مَعَ (ساتھ ہونا — Company)

**مَعَ** = کے ساتھ (ساتھ جانا، ساتھ ہونا)

| عربی | ترجمہ |
|------|--------|
| خَرَجَ هَامِدٌ مَعَ خَالِدٍ | حامد خالد کے ساتھ نکلا |
| ذَهَبَ الطَّبِيبُ مَعَ الْمُهَنْدِسِ | ڈاکٹر انجینئر کے ساتھ گیا |
| مَعِي كِتَابٌ | میرے ساتھ کتاب ہے |
| مَعَكَ قَلَمٌ | تیرے ساتھ قلم ہے |

### عِنْدَ بمقابلہ مَعَ

| | مطلب | مثال |
|--|------|-------|
| عِنْدَ | ملکیت / گھر میں ہونا | عِنْدِي سَيَّارَةٌ = میرے پاس گاڑی ہے |
| مَعَ | ساتھ ہونا | مَعِي سَيَّارَةٌ = میرے ساتھ گاڑی ہے (ابھی) |

---

## لِي بمقابلہ عِنْدِي

| | فرق | مثال |
|--|-----|-------|
| لِي أَخٌ | تعلق — میرا بھائی ہے | لِي أَخٌ وَاحِدٌ = میرا ایک بھائی ہے |
| عِنْدِي كِتَابٌ | ملکیت — میرے پاس ہے | عِنْدِي كِتَابٌ = میرے پاس کتاب ہے |

> **قاعدہ:** جو چیز آپ کی ملکیت ہو — عِنْدِي
> جس سے آپ کا تعلق ہو (رشتہ دار وغیرہ) — لِي

---

## مُنْصَرِف اور غیر مُنْصَرِف

### تعریف

| قسم | تعریف | علامت |
|-----|-------|-------|
| مُنْصَرِف | جس اسم پر تنوین آئے | دو پیش / دو زبر / دو زیر |
| غیر مُنْصَرِف | جس اسم پر تنوین نہ آئے | صرف ایک حرکت |

### مُنْصَرِف کی مثالیں (تنوین کے ساتھ)

مُحَمَّدٌ، خَالِدٌ، حَامِدٌ، عَبَّاسٌ، مَحْمُودٌ

### غیر مُنْصَرِف کی مثالیں (بغیر تنوین)

| اسم | وجہ |
|-----|-----|
| أَحْمَدُ | وزنِ فعل + علَمیت |
| عُمَرُ | علَمیت + عجمی |
| فَاطِمَةُ | گول تا + علَمیت |
| طَلْحَةُ | گول تا + علَمیت |
| زَيْنَبُ | علَمیت + مؤنث |
| إِبْرَاهِيمُ | عجمی نام |

> **آسان قاعدہ:** عورتوں کے اکثر نام اور بہت سے مردوں کے نام (جو خاص وزن پر ہوں) غیر مُنْصَرِف ہوتے ہیں۔

### فرق صرف پڑھنے میں

| | مُنْصَرِف | غیر مُنْصَرِف |
|--|---------|------------|
| رَفْع | مُحَمَّدٌ | أَحْمَدُ |
| نَصْب | مُحَمَّدًا | أَحْمَدَ |
| جَر | مُحَمَّدٍ | أَحْمَدَ |

> **نوٹ:** غیر مُنْصَرِف میں نصب اور جر ایک جیسے ہوتے ہیں (دونوں میں زبر)

## ضمائر کا مکمل نظام

### منفصل ضمائر (Separate)

| ضمیر | مطلب |
|------|------|
| أَنَا | میں |
| أَنْتَ | تم (مذکر) |
| أَنْتِ | تم (مؤنث) |
| هُوَ | وہ (مذکر) |
| هِيَ | وہ (مؤنث) |
| هُمَا | وہ دونوں |
| هُمْ | وہ (جمع مذکر) |
| هُنَّ | وہ (جمع مؤنث) |

## سبق ۱۰ کے جملے

| عربی | ترجمہ |
|------|--------|
| وَالِدِي فِي الْكُوَيْتِ | میرے والد کویت میں ہیں |
| هُوَ طَبِيبٌ شَهِيرٌ | وہ مشہور ڈاکٹر ہیں |
| أَخِي مَعِي هُنَا فِي الْمَدِينَةِ الْمُنَوَّرَةِ | میرا بھائی یہاں مدینہ منورہ میں میرے ساتھ ہے |
| لِي أُخْتٌ وَاحِدَةٌ اسْمُهَا زَيْنَبُ | میری ایک بہن ہے جس کا نام زینب ہے |
| هِيَ فِي الْعِرَاقِ مَعَ زَوْجِهَا | وہ اپنے شوہر کے ساتھ عراق میں ہے |
| زَوْجُهَا مُهَنْدِسٌ | اس کا شوہر انجینئر ہے |
| مَا عِنْدِي سَيَّارَةٌ | میرے پاس گاڑی نہیں |
| عِنْدَ أَخِي سَيَّارَةٌ | میرے بھائی کے پاس گاڑی ہے |
| خَرَجَ وَالِدِي مِنَ الْبَيْتِ | میرے والد گھر سے نکلے |
| مَعَهُ عَمِّي | ان کے ساتھ میرے چچا ہیں |
| أَخِي ذَهَبَ إِلَى الْمَدْرَسَةِ | میرا بھائی اسکول گیا |
| وَالِدِي ذَهَبَ إِلَى الْجَامِعَةِ | میرے والد یونیورسٹی گئے |

## کَانَ — ماضی کا اظہار

**كَانَ** = تھا / تھی (Verb to be in past)

| عربی | ترجمہ |
|------|--------|
| كَانَ تَاجِراً | وہ تاجر تھے |
| كَانَ أَبِي تَاجِراً | میرے ابا تاجر تھے |

> **نوٹ:** كَانَ کے بعد اسم نصب میں آتا ہے

## مشق (تَمَارِيْن)

### مشق الف — متصل ضمیر لگائیں

| اسم | میرا | تیرا | اس کا | اس کی |
|-----|------|------|-------|-------|
| قَلَمٌ | قَلَمِي | قَلَمُكَ | قَلَمُهُ | قَلَمُهَا |
| بَيْتٌ | _______ | _______ | _______ | _______ |
| سَيَّارَةٌ | _______ | _______ | _______ | _______ |
| أُخْتٌ | _______ | _______ | _______ | _______ |

### مشق ب — عِنْدَ یا لِي؟

| جملہ | صحیح انتخاب |
|------|------------|
| میرا ایک بھائی ہے | لِي أَخٌ |
| میرے پاس کتاب ہے | عِنْدِي كِتَابٌ |
| اس کے پاس گاڑی ہے | عِنْدَهُ سَيَّارَةٌ |
| اس کی ایک بہن ہے | لَهَا أُخْتٌ |

### مشق ج — مُنْصَرِف یا غیر مُنْصَرِف؟

| نام | قسم |
|-----|-----|
| مُحَمَّدٌ | مُنْصَرِف |
| فَاطِمَةُ | غیر مُنْصَرِف |
| خَالِدٌ | مُنْصَرِف |
| زَيْنَبُ | غیر مُنْصَرِف |
| حَامِدٌ | مُنْصَرِف |
| إِبْرَاهِيمُ | غیر مُنْصَرِف |

## ہوم ورک

- تمام مشقیں **لکھ کر** مکمل کریں
- اپنے خاندان کے بارے میں ۱۰ جملے بنائیں (والدین، بھائی بہن)
- پانچ اسماء پر چاروں متصل ضمائر لگائیں
- مُنْصَرِف اور غیر مُنْصَرِف ناموں کی الگ الگ فہرست بنائیں

## آج کا خلاصہ

| موضوع | اہم نکتہ |
|-------|----------|
| متصل ضمیر | ی (میرا) — كَ (تیرا) — هُ/هَا (اس کا/کی) |
| لامِ متصل | لِي / لَكَ / لَهُ — ملکیت یا تعلق |
| عِنْدَ | کے پاس — ملکیت |
| مَعَ | کے ساتھ — صحبت |
| لِي بمقابلہ عِنْدِي | تعلق بمقابلہ ملکیت |
| مُنْصَرِف | تنوین آتی ہے (~۸۵٪ اسماء) |
| غیر مُنْصَرِف | تنوین نہیں آتی — نصب اور جر ایک جیسے |
| كَانَ | ماضی کا فعل — بعد میں نصب |
